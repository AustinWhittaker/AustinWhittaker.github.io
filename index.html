The Diamond of Florentine

This Diamond of Florentine is Virtual Reality game that takes place in the 17th century.
Within this game the player goes on a quest to retrieve the lost Florentine diamond from the bottom
of the ocean before it becomes lost forevermore. This game was created with the use of Unity within
a 3 month time span. I was the Project and lead developer for this game with coproducers Alina Krause,
and Cole Roberts. This game was created at the University of North Carolina within the Topics in Computer
Science class. Please enjoy the trailer that we created.

I was responsible for the creation of the introduction with included the main menu, story building, and
boat battle cutscene. Within the main menu the player can choose to begin a new game or (for demo purposes)
choose any scene they would like to begin. The user also has the ability to change the dynamics of the game
such as whether they prefer snap turn or continuous turning (incase of motion sickness) This game has high
definition graphics that are too advanced for an apk format for the Oculus Quest 2 to download onto sidequest.
This video will demonstrate timelines, object oriented programming, C# scripting for cannon firing, and the use of
Blender for character creation.

On top of the previous video I also created swimming functionality for a user to swim within the ocean to look
for clues on the diamonds whereabouts. This functionality was possible by obtaining the velocity and the location of
the Oculus Quest 2 controllers. It is a grab and release method to pull the player with force and slow them down
with drag. This video also demonstrates an air supply function that keeps the player on a timer so that if they
run out of air the game will come to an end. Once the player returns to the surface their air supply will reset.
